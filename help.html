Video Cleaner Help

Video Cleaner is an application designed to remove extraneous bytes from MP4, MOV, and WMV video files.

Quick Start Guide

  1. Add video files by dragging and dropping them into the app or using the File menu.
  2. Add entire directories with drag-and-drop.
  3. Click the "Clean files" button to remove extra bytes from the added files.
  4. Check the status of each file in the list.
  5. Use "Remove clean from list" to clear processed files.

Adding Files

  • Drag and drop files or folders directly into the app window.
  • Use the File menu and select "Open" (or press Ctrl+O).
  • The app supports MP4, MOV, and WMV video files.

Removing Files

  • Click "Remove clean from list" to clear only the processed files.
  • Use "Clear File List" from the File menu to remove all files.
  • Press Ctrl+R or use "Reload" from the View menu to completely reset the app.

Saving Removed Extra Bytes

Check the "Save removed extra bytes" box to create a .txt file next to each processed video file. This file will contain the bytes that were removed during cleaning.

Additional Information

  • The app processes files in the order they were added.
  • You can add more files at any time, even when there are files already in the list.
  • The status indicator shows whether a file is pending, cleaned, or encountered an error.
  • Hover over the status indicator to see more details about any errors or extra bytes found.
  • The total file count is displayed at the bottom of the main window.
